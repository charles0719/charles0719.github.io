注意：段落标题必须使用中文（技能清单、工作经历、项目经历等），脚本通过这些关键词来识别各个段落。

工作原理

  1. 解析：用正则表达式提取 Markdown 中的姓名、技能、工作经历、项目经历等结构化数据
  2. 模板填充：将解析出的数据动态填入 HTML 模板，不受模板占位符数量限制
  3. PDF 渲染：通过 Puppeteer 启动无头 Chrome，渲染 HTML 并导出 A4 尺寸的 PDF

常见问题

问题 解决方案
Cannot find package 'puppeteer-core' 进入 scripts/ 目录执行 npm install
Chrome/Chromium not found 安装 Chrome：brew install --cask google-chrome
PDF 内容被截断 检查 Markdown 格式是否符合规范，尤其是段落标题
生成后想微调排版 先打开生成的 .html 文件预览，调整满意后再生成 PDF
